# IPL SQL Analysis

This repository contains a SQL-based data analysis project on IPL (Indian Premier League) datasets. The project focuses on extracting insights using structured SQL queries.

---

## 📌 Project Overview

The objective of this project is to analyze IPL matches and deliveries data using SQL to uncover patterns in player performance, team results, and match trends.

---

## 📂 Project Structure

### 🔹 SQL Queries
- Pre-written SQL queries for:
  - Player performance analysis
  - Team statistics
  - Match trends and insights

### 🔹 Dataset
- **Matches Table**: Match-level details (season, teams, winner, venue)
- **Deliveries Table**: Ball-by-ball data (batsman, bowler, runs, dismissals)

---

## 🎯 Objectives

- Analyze player performance across seasons  
- Identify top-performing batsmen and bowlers  
- Evaluate team performance and winning patterns  
- Study venue-based scoring trends  

---

## 📊 Key Analysis

- Player statistics (runs, wickets, strike rates, averages)  
- Season-wise top performers  
- Team win/loss analysis  
- Venue-based performance insights  

---

## 💡 Insights

- Identified consistent high-performing players  
- Observed scoring patterns across venues  
- Analyzed team performance trends across seasons
